u16javall
===============

1. openjdk7
2. openjdk8
3. oraclejdk7
4. oraclejdk8

1. elasticsearch 5.1.2
2. memcached 1.4
3. mongodb 3.4
4. mysql 5.7
5. neo4j 3.1.1
6. postgres 9.4
7. rabbitmq 3.6.6
8. redis 3.2
9. selenium 3.0
10. sqllite 3
11. couchdb 2.0
12. rethinkdb 2.3
13. riak 2.2
14. cassandra 3.9
